data()

data(null|string|array  $arg = null, null|string  $val = null) : string|\PerryRylance\DOMElement

Method for working with data- attributes on this element

Parameters

null|string|array $arg

If both arguments are null / not provided, this function will return all data- attributes as an associative array. If a string is provided, it will be treated as a name and the value of the relevant data- attribute will be returned. If an array is provided, it will be used to set multiple data- attributes on the element.
null|string $val

A second argument, this can only be used if $arg is a string

Throws

\Exception

When $arg is null, but $val is non-null. This is an invalid combination of arguments

\Exception

When $arg is a string, but $val is not a string or null

\Exception

When $arg is an array of key value pairs to set, but $val is not null

\Exception

When the combination of arguments supplied is invalid, or one or more types in the arguments are invalid

Returns

string|\PerryRylance\DOMElement —

A string when retrieving data, this element when setting data
