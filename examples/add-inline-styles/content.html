Hi there!

This example shows off a small part of the DOMDocument library - query selectors and the 
css()
function. Using this PHP CSS Parser, we can obtain the rules from 
style.css
, select the elements using 
find()
and then apply the styles.
Internally we use Symfony's CSS selector to convert selectors to XPaths, which are then used to select elements server side.
The result is a beautifully inline-styled document. This could be useful, say, for a mailer wishing to send styled mails but still keep content and presentation separate.
This library has many more uses as a jQuery-like, server side DOM manipulation library. I hope it is useful to you. Contributions are most welcome.
